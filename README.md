## ⚡ Quick Setup

```bash
git clone https://github.com/hamidlabs/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```

## ⚙️ Customization

### Change Theme

```json
{
	"workbench.colorTheme": "Your Theme Name"
}
```

### Update Font

```json
{
	"editor.fontFamily": "Your Font, JetBrains Mono, Consolas"
}
```
